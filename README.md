# pwsbasictrivia-v1

React component that renders a PWS educational trivia game. It uses Tailwind-inspired styling and includes a comprehensive question bank across basic, intermediate and advanced levels. See `PWSTriviaGame.js` for the full component implementation.

## Usage

Import `PWSTriviaGame` into your React application and render it within your component tree. The component provides a setup screen, quiz experience and results view.
